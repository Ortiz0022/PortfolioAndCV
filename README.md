#  Portfolio & CV Web

Portafolio web personal desarrollado para presentar proyectos, habilidades y experiencia académica en Ingeniería en Sistemas.  
La aplicación permite visualizar proyectos destacados, tecnologías utilizadas y acceder a más información mediante un diseño moderno, interactivo y adaptable a modo claro y oscuro.

---

##  Descripción

Este proyecto consiste en un sitio web tipo **portfolio profesional** que reúne información relevante sobre mi perfil como desarrolladora.  
Incluye una sección de presentación, portafolio de proyectos, tecnologías utilizadas y enlaces externos para explorar cada proyecto con más detalle.

El objetivo principal es mostrar de forma clara y visual los proyectos desarrollados durante mi formación en Ingeniería en Sistemas, así como las tecnologías utilizadas en cada uno.

---

##  Tecnologías utilizadas

El proyecto fue desarrollado utilizando tecnologías modernas de desarrollo web:

- **React** – Construcción de la interfaz de usuario
- **TypeScript** – Tipado estático para mayor robustez
- **Vite** – Herramienta de desarrollo y build rápida
- **TailwindCSS** – Estilizado moderno y responsivo
- **Shadcn UI** – Componentes accesibles y reutilizables
- **Lucide Icons** – Íconos para la interfaz
- **Vercel** – Despliegue y hosting del proyecto

---

##  Características principales

- Diseño moderno y minimalista
- Animaciones suaves e interacciones visuales
- Modo **claro y oscuro**
- Cards interactivas para proyectos
- Modal de detalle para cada proyecto
- Diseño **responsive** adaptable a diferentes dispositivos
- Fondo interactivo con estilo espacial
